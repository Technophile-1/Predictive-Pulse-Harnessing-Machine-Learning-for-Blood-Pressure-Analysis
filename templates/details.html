Blood Pressure Analysis System - Details

About the System

This Blood Pressure Analysis System uses machine learning algorithms to predict blood pressure stages based on various health parameters. The system analyzes multiple factors to provide accurate predictions about hypertension stages.

Input Features

  • Gender: Patient's gender (Male/Female)
  • Age: Patient's age in years
  • History: Family history of hypertension
  • Patient Type: Classification of patient type
  • Take Medication: Whether patient is on medication
  • Severity: Severity level of symptoms
  • Breath Shortness: Presence of shortness of breath
  • Visual Changes: Any visual disturbances
  • Nose Bleeding: Occurrence of nosebleeds
  • When Diagnosed: Time since diagnosis
  • Systolic Pressure: Systolic blood pressure reading
  • Diastolic Pressure: Diastolic blood pressure reading
  • Controlled Diet: Whether patient follows controlled diet

Prediction Categories

  • NORMAL: Blood pressure within normal range
  • HYPERTENSION (Stage-1): Mild hypertension
  • HYPERTENSION (Stage-2): Moderate to severe hypertension
  • HYPERTENSIVE CRISIS: Severe hypertension requiring immediate attention

Machine Learning Models Used

The system employs multiple machine learning algorithms:

  • Random Forest Classifier: Primary model for predictions
  • Logistic Regression: Linear classification approach
  • Decision Tree: Rule-based classification
  • Naive Bayes: Probabilistic classification

How to Use

  1. Fill in all the required health parameters in the form
  2. Ensure all values are accurate and within valid ranges
  3. Click "Predict Blood Pressure Stage" to get the prediction
  4. Review the predicted blood pressure stage
  5. Consult with healthcare professionals for medical advice
